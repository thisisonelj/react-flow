## 项目概述

### 项目简介

订单工作流审批系统是一个基于React的前端应用，用于管理和处理订单的审批流程。系统支持多级审批、流程自定义、审批记录追踪等功能，帮助企业实现订单审批的数字化和自动化。

### 核心功能

- **订单管理**：订单的创建、查看、编辑和删除
- **工作流配置**：自定义审批流程和审批节点
- **审批处理**：多级审批、审批意见填写、审批历史记录
- **权限管理**：基于角色的权限控制
- **数据统计**：审批效率分析、订单统计报表

### 业务流程

```
订单创建 → 提交审批 → 一级审批 → 二级审批 → ... → 审批完成 → 订单生效
                ↓
            审批拒绝 → 退回修改 → 重新提交
```

## 数据流设计

### 状态管理架构

推荐使用 Zustand 进行状态管理：

```typescript
// store/orderStore.ts
import { create } from 'zustand';
import { devtools, persist } from 'zustand/middleware';
import type { Order, OrderListParams } from '@/types/order';
import { orderApi } from '@/services/order';

interface OrderState {
  orders: Order[];
  currentOrder: Order | null;
  loading: boolean;
  total: number;
  params: OrderListParams;
  
  fetchOrders: (params: OrderListParams) => Promise<void>;
  fetchOrderDetail: (id: string) => Promise<void>;
  createOrder: (data: Partial<Order>) => Promise<void>;
  updateOrder: (id: string, data: Partial<Order>) => Promise<void>;
  deleteOrder: (id: string) => Promise<void>;
  setParams: (params: Partial<OrderListParams>) => void;
}

export const useOrderStore = create<OrderState>()(
  devtools(
    persist(
      (set, get) => ({
        orders: [],
        currentOrder: null,
        loading: false,
        total: 0,
        params: {
          page: 1,
          pageSize: 20,
        },

        fetchOrders: async (params) => {
          set({ loading: true });
          try {
            const response = await orderApi.getList(params);
            set({ 
              orders: response.list, 
              total: response.total,
              params 
            });
          } catch (error) {
            console.error(error);
          } finally {
            set({ loading: false });
          }
        },

        fetchOrderDetail: async (id) => {
          set({ loading: true });
          try {
            const order = await orderApi.getDetail(id);
            set({ currentOrder: order });
          } catch (error) {
            console.error(error);
          } finally {
            set({ loading: false });
          }
        },

        createOrder: async (data) => {
          set({ loading: true });
          try {
            await orderApi.create(data);
            const { params } = get();
            await get().fetchOrders(params);
          } catch (error) {
            console.error(error);
          } finally {
            set({ loading: false });
          }
        },

        updateOrder: async (id, data) => {
          set({ loading: true });
          try {
            await orderApi.update(id, data);
            const { params } = get();
            await get().fetchOrders(params);
          } catch (error) {
            console.error(error);
          } finally {
            set({ loading: false });
          }
        },

        deleteOrder: async (id) => {
          set({ loading: true });
          try {
            await orderApi.delete(id);
            const { params } = get();
            await get().fetchOrders(params);
          } catch (error) {
            console.error(error);
          } finally {
            set({ loading: false });
          }
        },

        setParams: (params) => {
          set({ params: { ...get().params, ...params } });
        },
      }),
      { name: 'order-storage' }
    )
  )
);
```

```
用户操作 → Action → API请求 → Store更新 → 组件重新渲染
```

### 缓存策略

使用 React Query 进行数据缓存：

```typescript
// hooks/useOrder.ts
import { useQuery, useMutation, useQueryClient } from '@tanstack/react-query';
import { orderApi } from '@/services/order';

export const useOrderList = (params) => {
  return useQuery({
    queryKey: ['orders', params],
    queryFn: () => orderApi.getList(params),
    staleTime: 5 * 60 * 1000, // 5分钟
  });
};

export const useCreateOrder = () => {
  const queryClient = useQueryClient();
  
  return useMutation({
    mutationFn: orderApi.create,
    onSuccess: () => {
      queryClient.invalidateQueries({ queryKey: ['orders'] });
    },
  });
};
```

#### 父子组件通信

```typescript
// 父组件
const Parent = () => {
  const [data, setData] = useState({});
  
  return <Child data={data} onChange={setData} />;
};

// 子组件
interface ChildProps {
  data: object;
  onChange: (data: object) => void;
}

const Child: React.FC<ChildProps> = ({ data, onChange }) => {
  return <div>{/* 组件内容 */}</div>;
};
```

#### 跨组件通信

使用 Context 或状态管理：

```typescript
// contexts/OrderContext.tsx
import { createContext, useContext } from 'react';

interface OrderContextType {
  currentOrder: Order | null;
  setCurrentOrder: (order: Order | null) => void;
}

const OrderContext = createContext<OrderContextType | null>(null);

export const useOrderContext = () => {
  const context = useContext(OrderContext);
  if (!context) {
    throw new Error('useOrderContext must be used within OrderProvider');
  }
  return context;
};
```
